Términos y Condiciones de Uso

Bienvenido a BugSolutions Project. Al utilizar esta aplicación, aceptas cumplir con los siguientes términos y condiciones. Si no estás de acuerdo, por favor no uses la plataforma.

1. Uso de la aplicación

Este sistema está destinado únicamente para fines académicos y administrativos. El usuario se compromete a usar los servicios de manera responsable.

2. Inicio de sesión con Facebook

Se utiliza Facebook Login únicamente para obtener información básica necesaria para autenticación. No publicamos ni interactuamos con tu perfil.

3. Privacidad de los datos

No almacenamos, vendemos o compartimos información personal con terceros. Los datos se usan solo para el acceso dentro de la aplicación.

4. Limitación de responsabilidad

No somos responsables de fallos técnicos, interrupciones del servicio o mal uso de la aplicación por parte de los usuarios.

5. Cambios en los términos

Estos términos pueden actualizarse ocasionalmente. La última versión siempre estará disponible en esta página.
